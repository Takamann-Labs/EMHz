## Overview

**EMHz** is a professional Android audio DSP application built for audiophiles. It features a proprietary multi-layer bass synthesis engine, studio-grade reverb, and spatial audio processing — all wrapped in a premium per-mode UI.

> **Note:** This repository distributes the compiled APK only. The DSP core and source code are proprietary and not publicly available.

---

## Modes

| Mode | Description |
|------|-------------|
|  **Sub-Atomic** | Multi-layer bass synthesis. Generates sub-harmonic depth beyond your source audio. |
|  **Studio** | Schroeder reverb architecture. Room acoustics, spatial imaging, stereo widening. |
|  **RAW Signal** | Unfiltered unprocessed audio. |
Each mode has its own visual identity — color, glow, and layout built to match the sonic character.

---

## 📲 Installation

1. Download the latest APK 
2. Enable *Install from unknown sources* on your Android device
3. Install and open EMHz 

**Minimum:** Android 8.0 (API 26) · **Recommended:** Android 11+
